HTML - Hypertext Markup Language

Think of HTML as the 'frame of a house. It contains all of the text, images, and other visible elements on a web page.

HTML Tag Example:

<h4>The text between the opening and closing tags will be displayed on your web page, as shown below.</h4>

This is what the text looks like when displayed on your web page.

    Anchor tags

  • The tag below is known as an 'anchor' tag and creates a link to another page.
  • <a href="https://google.com">Go to Google</a>
  • This is a working link. Click on it to see what happens.https://google.com
  • The 'href' is called an HTML attribute and tells the browser where to go.

Image tags

Image tags allow you to put pictures on your page. The image address is placed in between quotation marks.

<img src="https://images.unsplash.com/photo...blahblahblah" />

The 'src' is another HTML attribute and tells the browser where to access the image.
